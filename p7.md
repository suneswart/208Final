### Phase 7: Inheritance

Here are two ways to use JavaScript inheritance: [9]

```js
<!DOCTYPE HTML>
<html>
  <script>
    
    var o = {
        val: 5,
        getVal: function(){
             return this.val;
        }
    };

    var p = Object.create(o); //p inherits o

    document.write(p.getVal());
    
  </script>
</html>

```

Prints:

	5

Another way of using JavaScript inheretance:

```js
<!DOCTYPE HTML>
<html>
  <script>
    "use strict";
    
    class O{
        constructor(value){
            this.val = value;
        }
        getVal(){
            return this.val;
        }
    }

    class P extends O{
        constructor(value){
            super(value);
        }
    }

    var p = new P(5);

    document.write(p.getVal());
    
  </script>
</html>

```

Prints:

	5
